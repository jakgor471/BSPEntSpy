Search HELP

Find

Finds and selects the next matching entity relative to the start of selection or selected entity. Hold Shift to add all matched entities to a selection.



Filter

Filters the entity list, showing only the entities matching the criteria. Hold Shift to remove the filter.

The entities can be found or filtered by inputting a search-string into the search bar. By default the entity is matched if and only if a classname or targetname is equal to or contains a search-string.

Search by criterion syntax HELP

There is an option to use advanced search, this way it is possible to precisely specify the searched key-values of an entity.

Example:

$"targetname" = "zombie1"; "squadname" = "ravenholm_zombies";

Advanced search-string must start with '$' followed by one or more criteria separated by ';'

Criteria format:
"key" *OPERATION* *VALUE*;

Operations:

  • '=' - followed by "value", matches if key is equal to value.

  • 'like' - followed by "regex", matches if regex matches the key. See "Java Pattern".

  • 'in' - followed by ("value1", "value2", ..., "valueN") , matches if key is equal to one or more values.


Place holder format: %[key]

Place holders can be used in conjunction with any operation.



Examples:

Match all entities parented to themselves (parentname == targetname) using place holders:

$"parentname" = "%[targetname]";

Match all entities with targetname equal to "zombie" of which class is either "npc_zombie", "npc_fastzombie" or "npc_poisonzombie":

$"targetname" = "zombie"; "classname" in ("npc_zombie", "npc_fastzombie", "npc_poisonzombie");

Match all entities of which "model" parameter starts with "*" (brush entities):

$"model" like "\*.*";

Explanation: "\*" means "match '*' character", ".*" means "any character, zero or more times".


Search in radius syntax HELP

It is possible to filter entities by position (search in radius). Such search-string must start with '@' followed by position, comma (',') and radius:

Example:

@128 -32 65536, 128;

Matches entities within 128 unit radius around (128, -32, 65536)
