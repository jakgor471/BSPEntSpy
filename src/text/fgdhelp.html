FGD files

FGD files (Game Definition File) are used by Hammer Editor to display additional information about the entities. It is possible to load a single FGD file by clicking File > Load FGD File. Once the FGD is loaded the Smart Edit mode can be enabled. In Smart Mode the entity parameter names are translated to their respective display names.

Once the FGD file has been successfully loaded the program will try to automatically load it on subsequent start. If remembered file cannot be opened or an error occurs during reading - it will be forgotten.

FGD files can be found in <SteamApps>/common/GAME/bin directories.

Commonly used FGDs:

  • Half-Life 2 and Episodes: <SteamApps>/common/Half-Life 2/bin/halflife2.fgd
  • Garry's Mod: <SteamApps>/common/GarrysMod/bin/garrysmod.fgd
  • Counter-Strike: Source: <SteamApps>/common/Counter-Strike Source/bin/cstrike.fgd
  • Half-Life 1 related: <SteamApps>/common/Half-Life SDK/Hammer Editor/fgd/
